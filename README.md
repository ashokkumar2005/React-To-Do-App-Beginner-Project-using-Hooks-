# React To-Do App

This is a simple To-Do App built using React.  
This project is created to understand the basic concepts of React step by step.

---

## Project Description

In this project, users can add tasks and delete tasks.  
The main purpose of this project is to learn how React works using a small real-world example.

---

## Features

- Add a new task
- Delete a task
- Show tasks in a list
- Show message when no tasks are available
- Beginner friendly project

## How It Works

1. The user types a task in the input box
2. The task is stored using useState
3. When the user clicks the Add button, the task is added to the list
4. React re-renders the component and shows the task
5. The user can delete the task using the Delete button
